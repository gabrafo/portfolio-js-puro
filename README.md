# Portfólio Pessoal (Bootstrap e Tailwind)

Este projeto tem duas versões da mesma página pessoal:

- versao-bootstrap
- versao-tailwind

Ambas possuem:

- layout responsivo
- tema claro e tema escuro
- botão de alternância de tema com JavaScript

A intenção é que seja um site simples, não sendo meu portfólio definitivo.

## Como rodar

1. Entre na pasta da versão desejada.
2. Abra o arquivo index.html no navegador.

Arquivos principais:

- versao-bootstrap/index.html
- versao-tailwind/index.html
